The Categorical Variable Distribution module assesses the presence and frequency of categorical values in a column of a dataset. For example, it can display the spectrum of units associated with a lab or medication.

## Installation

You can install the development version of this package like so:

``` r
devtools::install_github('ssdqa/categoricalvariabledistribution')
```
