# Multi Site, Exploratory, Longitudinal Output for Categorical Variable Distribution Module

Multi Site, Exploratory, Longitudinal Output for Categorical Variable
Distribution Module

## Usage

``` r
cvd_ms_exp_la(process_output)
```

## Arguments

- process_output:

  *tabular output* \| The output from `cvd_process`

## Value

A line plot, returned as a ggplot object, faceted by categorical
valueset item, with lines and line color representing site.
