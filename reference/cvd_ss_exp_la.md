# Single Site, Exploratory, Longitudinal Output for Categorical Variable Distribution Module

Single Site, Exploratory, Longitudinal Output for Categorical Variable
Distribution Module

## Usage

``` r
cvd_ss_exp_la(process_output)
```

## Arguments

- process_output:

  *tabular output* \| The output from `cvd_process`

## Value

A line plot as a ggplot object with time on x axis, proportion of
category with value on y axis, colored by categorical value
