# Single Site, Exploratory, Cross-Sectional Output for Categorical Variable Distribution Module

Single Site, Exploratory, Cross-Sectional Output for Categorical
Variable Distribution Module

## Usage

``` r
cvd_ss_exp_cs(process_output)
```

## Arguments

- process_output:

  *tabular output* \| The output from `cvd_process`

## Value

a bar plot as a ggplot object with categorical value set on y axis,
proportion on y axis, filled by categorical value
