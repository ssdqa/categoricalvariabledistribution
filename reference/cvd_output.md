# Categorical Variable Distribution - Output Generation

Using the tabular output generated by `cvd_process`, this function will
build a visualization of the results Each function configuration will
output a bespoke ggplot object. Plot theming can be adjusted by the user
after the graph has bee output using `+ theme()`. Plots can also be made
interactive by using `make_interactive_squba()` on the object output
from `cvd_output`.

## Usage

``` r
cvd_output(process_output, filter_concept, vocab_tbl = NULL)
```

## Arguments

- process_output:

  *tabular output* \| The output from `cvd_process`

- filter_concept:

  *numeric/string or vector* \| The specific code(s) that should be the
  focus of the analysis; used for `ss_anom_la` and `ms_anom_la`

- vocab_tbl:

  *tabular input* \| OPTIONAL: the location of an external vocabulary
  table containing concept names for the provided codes. if not NULL,
  concept names will be available in either a reference table or in a
  hover tooltip

## Value

a graph or table to visualize the results from
