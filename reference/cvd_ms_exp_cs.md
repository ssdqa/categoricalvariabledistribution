# Multi Site, Exploratory, Cross-Sectional Output for Categorical Variable Distribution Module

Multi Site, Exploratory, Cross-Sectional Output for Categorical Variable
Distribution Module

## Usage

``` r
cvd_ms_exp_cs(process_output)
```

## Arguments

- process_output:

  *tabular output* \| The output from `cvd_process`

## Value

A heatmap, returned as a ggplot object, with site on x axis, valueset
item on y axis, with proportion of the site's total for each valueset
item displayed as tile color and a label on the tile
